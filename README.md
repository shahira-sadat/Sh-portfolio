## :running: Run Locally

Clone the project

```bash
  git@github.com:shahira-sadat/Sh-portfolio.git
```

Go to the project directory

```bash
  cd Sh-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
